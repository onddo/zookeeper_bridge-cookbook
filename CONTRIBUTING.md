Contributing
============

1. [Fork the repository on Github](https://help.github.com/articles/fork-a-repo).
2. Create a named feature branch (`$ git checkout -b my-new-feature`).
3. Write your change.
4. Write tests for your change (if applicable).
5. [Run the tests](https://github.com/zuazo/zookeeper_bridge-cookbook/blob/master/TESTING.md), ensuring they all pass (`$ bundle exec rake`).
6. Commit your change (`$ git commit -am 'Add some feature'`).
7. Push to the branch (`$ git push origin my-new-feature`).
8. [Submit a Pull Request using Github](https://help.github.com/articles/creating-a-pull-request).

Please do not hesitate to [open an issue](https://github.com/zuazo/zookeeper_bridge-cookbook/issues/new) with any questions or problems.
